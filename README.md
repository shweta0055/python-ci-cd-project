**🚀 Python CI/CD Project using GitHub Actions**

**📌 Overview**

This project demonstrates the implementation of Continuous Integration and Continuous Deployment (CI/CD) using GitHub Actions for a Python-based web application.

It includes automated testing, Docker containerization, and workflow automation to ensure code quality and seamless deployment.

**🛠️ Tech Stack**

Python

Flask

Pytest

Docker

GitHub Actions

**⚙️ Features**

✅ REST API using Flask

✅ Automated testing using Pytest

✅ CI pipeline using GitHub Actions

✅ Docker containerization

✅ Automated build on every push
